# Tomato 
Manage your WP boxes like a tomato.

## Built using
* Django

## Installation
* `sudo pip install -r requirements.txt`
* `./manage.py migrate`

## Quick guide
**Add migration code**: `./manage.py makemigrations`  
**Migrate database**: `./manage.py migrate`  
**Run server**: `./manage.py runserver`
